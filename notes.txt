In Windows Command Prompt, you can use the set command to temporarily set the
 environment variables before running the go build command. Here's an example:

bash
Copy code


set GOOS=linux
set GOARCH=amd64
go build -o output_name

// back to Windows
set GOOS=windows
set GOARCH=amd64
